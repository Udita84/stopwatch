-> Stopwatch Web Application

A simple and responsive stopwatch with **HTML, CSS and JavaScript**. With this application, you can measure elapsed time with precision and simplicity.

* Shows hours, minutes, seconds, milliseconds
* Responsive design for mobile and desktop
* User interface is intuitive and easy to use


-> How to run
1. Clone the repo:
   --bash git clone-- https://github.com/Udita84/stopwatch.git
3. Access the project folder:
   --bash cd stopwatch--
5. Open `index.html` in your favorite web browser.

No other software or dependencies required.
To start, simply open the application in your browser and use the simple and responsive interface to start, pause and reset the stopwatch.
